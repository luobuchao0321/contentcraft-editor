# Launch Posts

## Core Positioning

不是 Markdown 编辑器，而是公众号 SVG 排版模块复用工具。

ContentCraft can import WeChat articles, extract SVG layout modules, replace images safely, and build a local reusable content library.

## English

I built an open-source tool to extract reusable SVG layout blocks from WeChat articles.

Unlike Markdown-based WeChat editors, ContentCraft focuses on importing existing WeChat articles, extracting SVG layout modules, replacing images safely, and building a local-first content library.

GitHub: https://github.com/luobuchao0321/wechat-article-editor

## 中文

我做了一个开源公众号 SVG 排版复用工具。

它不是 Markdown 编辑器，而是解决“好看的公众号排版不能复用”的问题：复制一篇文章，提取 SVG 模块，替换图片，保存到自己的本地素材库，再用于新文章。

GitHub: https://github.com/luobuchao0321/wechat-article-editor

## 7-Day Promotion Plan

- Day 1: 发布定位：开源公众号 SVG 排版复用工具。
- Day 2: 发 30 秒动图：复制公众号文章，自动拆模块。
- Day 3: 技术文章：为什么复制到后台容易丢样式，如何转 inline HTML。
- Day 4: 差异对比：Markdown 编辑器 vs SVG 排版复用工具。
- Day 5: 桌面版：本地优先，不上传文章内容。
- Day 6: 教程：如何搭建自己的公众号素材库。
- Day 7: Roadmap：AI 清洗、模板包、批量换图，征集反馈。
