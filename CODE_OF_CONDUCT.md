# Code of Conduct

ContentCraft follows a simple rule: be practical, respectful, and constructive.

Please:

- Discuss ideas and tradeoffs, not personal attacks.
- Respect copyright and platform rules.
- Do not request or contribute paid/VIP template scraping.
- Avoid posting private drafts, API keys, cookies, or account credentials.

Maintainers may close or remove content that is abusive, unsafe, spammy, or outside the project scope.
