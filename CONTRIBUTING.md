# Contributing

Thanks for helping improve ContentCraft.

## Good First Contributions

- Add README screenshots or GIFs.
- Improve English documentation.
- Add copyright-safe example layout modules.
- Improve SVG block selection and image replacement tests.
- Add deployment notes for Vercel or Cloudflare Pages.

## Development

```bash
npm install
npm run dev
npm run build
```

Default local URL:

```text
http://localhost:3001
```

## Pull Request Checklist

- Keep sample assets synthetic or user-owned.
- Do not add paid/VIP template assets from third-party editors.
- Do not commit `.env`, API keys, cookies, `.next`, `.cache`, or local drafts.
- Run `npm run build` before submitting when code changes.

## Scope

ContentCraft focuses on WeChat article layout reuse, SVG module extraction, safe image replacement, local-first asset libraries, and clean inline HTML export.
