# ContentCraft Examples

These examples are synthetic, copyright-safe fixtures that demonstrate ContentCraft's core workflow:

1. Import a WeChat-style article.
2. Extract reusable SVG/layout blocks.
3. Replace images safely.
4. Reuse saved blocks in a new article.

No third-party WeChat article or paid template asset is included.

## Folders

- `sample-articles/` — small WeChat-style HTML articles for import testing.
- `svg-blocks/` — standalone SVG/layout modules.
- `wechat-layouts/` — reusable layout snippets.
- `before-after/` — before/after examples showing module reuse.

## Suggested Demo Flow

1. Open ContentCraft.
2. Paste `sample-articles/product-launch.html` into the import panel.
3. Extract modules and save the title block.
4. Insert `svg-blocks/hero-title.svg` into a new article.
5. Replace the image placeholder and copy WeChat-compatible HTML.
